Program Objective: This Database Management is intended for a operator who needs to manage race cars. This DMS will allow them to print all race cars to screen, add a new race car with details, remove a race car, update a race car entry, and launch/simulate a race. The initial startup will display a menu. You can then load a text file to fill the system with data. Once a file has been loaded, the menu will be return and allow for the user to perform the actions as mentioned above.

Note: You can download the RaceData.txt file as a starter data. It has 20 race cars already formatted. It is also included in the source folder altogether.
