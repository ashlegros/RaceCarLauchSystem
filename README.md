Program Objective: This Database Management System is intended for a operator who needs to manage race cars. This DMS will allow them to print all race cars to screen, add a new race car with details, remove a race car, update a race car entry, and launch/simulate a race.

Note: You can download the RaceData.txt file as a starter data. It has 20 race cars already formatted. It is also included in the source folder altogether.
Update: The progrom now connects to a sqlite database (db extension). The RaceCarDB.db is provided in the src folder.
